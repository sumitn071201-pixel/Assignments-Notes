GitHub Copilot: To write a regular expression on your own, follow these steps:

1. **Define your goal:** Decide what pattern you want to match (e.g., email, numbers, specific words).
2. **Break down the pattern:** Identify the components (letters, digits, special characters, repetitions).
3. **Use regex symbols:**
   - `^` = start of string
   - `$` = end of string
   - `.` = any character
   - `*` = zero or more
   - `+` = one or more
   - `?` = zero or one
   - `[abc]` = any of a, b, or c
   - `\\d` = digit
   - `\\w` = word character (letter, digit, underscore)
   - `\\s` = whitespace
   - `(abc)` = group
   - `{n,m}` = between n and m times

4. **Build and test:** Combine symbols to form your expression. Test it using online tools like [regex101.com](https://regex101.com/) or in your code.

**Example:**  
To match a 5-digit number:  
`/^\\d{5}$/`
